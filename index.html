Air Quality Analyzer - Mumbai

Project Overview

This dashboard provides insights into the air quality in Mumbai, displaying real-time and historical data trends for pollutants like AQI, PM2.5, NO2, CO, and more.

Live Power BI Dashboard

About This Project

Data Pipeline: Data is ingested from public APIs, processed in Azure Data Factory, stored in Azure SQL, and visualized using Power BI.

Technologies Used: Azure Blob Storage, Data Factory, SQL Database, Power BI, GitHub Pages
